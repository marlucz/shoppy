# Shoppy - Your private shopping list

Shopping list app with basic functionality based on VanillaJS and MDB 5 Webpack Starter

---

### Installation

```
npm install
```

### Dev Server

```
npm start
```

### Build

```
npm run build
```

### Files structure:

```
├── src
│   ├── img/
│   ├── js/
│   ├── scss/
│   └── index.html
├── webpack
│   ├── webpack.common.js
│   ├── webpack.config.dev.js
│   └── webpack.config.prod.js
└── dist/
```
